# 1st-c++-project
# 通讯录管理系统 (Communication System)

这是一个基于 C++ 开发的控制台通讯录管理系统，专为 macOS 终端环境优化。该项目旨在帮助用户实现联系人的增、删、改、查等日常管理需求，并针对初学者容易遇到的输入流错误进行了稳健性处理。

## 💡 核心功能亮点
- **联系人管理**: 支持添加、显示、删除、查找、修改和一键清空联系人记录。
- **工业级输入校验**: 使用 `std::numeric_limits` 和缓冲区清理技术，彻底解决了输入非法字符（如字母）导致的程序死循环问题。
- **交互友好**: 专为 macOS 终端优化，包含自动清屏逻辑和回车确认机制，界面清晰整洁。
- **数据处理安全**: 严谨的指针与结构体操作，确保程序运行的稳定性。

## 🛠 技术栈
- **编程语言**: C++
- **开发环境**: macOS Terminal, g++ 编译器
- **版本控制**: Git / GitHub
